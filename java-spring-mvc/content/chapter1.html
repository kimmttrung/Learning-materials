Spring Initializr - Giải thích giao diện

1. Project

Phần này cho phép bạn chọn kiểu dự án mà bạn muốn tạo:

  • Gradle - Groovy: Sử dụng Groovy làm ngôn ngữ cho Gradle.
  • Gradle - Kotlin: Sử dụng Kotlin làm ngôn ngữ cho Gradle.
  • Maven: Chọn Maven để quản lý và build dự án.

2. Spring Boot

Cho phép bạn chọn phiên bản của Spring Boot. Phiên bản ổn định hoặc snapshot có thể được chọn, ví dụ:

  • 3.4.2 (SNAPSHOT): Phiên bản thử nghiệm mới nhất.(Tốt nhất đừng ấn vào vì nó đang thử nghiệm)
  • 3.4.1: Phiên bản ổn định đã phát hành.

3. Project Metadata

Các thông tin cơ bản cho dự án của bạn, bao gồm:

  • Group: Tên nhóm tổ chức (thường theo định dạng tên miền ngược, ví dụ: com.example).
  • Artifact: Tên file đầu ra (ví dụ: demo).
  • Name: Tên dự án hiển thị.
  • Description: Mô tả ngắn gọn về dự án.
  • Package name: Tên package Java tự động được tạo.

4. Packaging

Lựa chọn cách đóng gói ứng dụng:

  • Jar: File thực thi độc lập.
  • War: File triển khai trên máy chủ ứng dụng (Tomcat, Jetty,...).

5. Java Version

Chọn phiên bản Java sẽ được sử dụng để xây dựng dự án. Các phiên bản phổ biến:

  • Java 17: Phiên bản LTS hiện tại, phù hợp cho các ứng dụng doanh nghiệp.
  • Java 21: Phiên bản mới với nhiều tính năng cải tiến.

6. Dependencies

Cho phép bạn thêm các thư viện hoặc module vào dự án, ví dụ: Spring Web, Spring Data JPA, Spring Security.

Nhấn nút Add Dependencies... để chọn các thư viện bạn cần.

7. Nút thao tác

  • Generate: Tải về dự án đã được tạo cấu trúc.
  • Explore: Xem trước dự án trực tuyến.

Hello World với Spring

Việc này đơn giản!! Sau khi tạo xong , mở dự án với VS Code(đợi 1 chút, cho IDE load project)

  1. Tạo file controller/HelloController.java như hình (cứ làm đi chương sau mình sẽ giải thích)

    Lưu ý: folder controller phải cùng cấp với file Application.java ấy nhé ( không tí lỗi đỏ mắt)

    code file HelloController.java :

      @Controller
  public class HelloController {
  
      @GetMapping("/")
      @ResponseBody
      public String requestMethodName() {
          System.out.println("ok");
          return "ok";
      }
  
  }

    Đồng thời hãy sửa file Application.java :

      package com.minh.demo;
  
  import org.springframework.boot.SpringApplication;
  import org.springframework.boot.autoconfigure.SpringBootApplication;
  
  // @SpringBootApplication
  @SpringBootApplication(exclude = org.springframework.boot.autoconfigure.security.servlet.SecurityAutoConfiguration.class)
  public class DemoApplication {
  
      public static void main(String[] args) {
          SpringApplication.run(DemoApplication.class, args);
      }
  
  }

  2. Các fen nghĩ chạy được rồi hả ? Không không :v Config nốt cái này

    Cài Đặt file :application.properties

    
              spring.mvc.trailing-slash-match=true
  
              # config database
  spring.datasource.url=jdbc:mysql://localhost:3306/your_database_name  #your_database_name : tên database
  spring.datasource.username=your_username  #username của db server 
  spring.datasource.password=your_password  #password
  spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
  spring.jpa.hibernate.ddl-auto=update

  3. Ok chạy thôi

    Cách 2: chúng ta sẽ chạy với file mvw (không cần tới IDE)

    .\mvnw spring-boot:run

    Thường thì lệnh trên sẽ chạy được cả Mac lẫn Win.Nếu win không chạy được hãy thử 

     .\mvnw.cmd spring-boot:run

  4. Truy cập localhost:8080/ nhé! :
