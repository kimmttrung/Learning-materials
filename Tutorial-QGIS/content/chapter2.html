Hướng dẫn thu thập mẫu dữ liệu trong QGIS

1. Chuyển đổi hệ tọa độ (Warp Raster)

- Gõ warp để mở hộp thoại Raster Projections.
- Target CRS: chọn UTM Zone 48N.
- Lưu kết quả trong file ggz: VLUCD_48N.
- Chuột phải lớp Ha Noi → chọn Export → Save Feature As → lưu file ở qpkg: HaNoi_48N.
- Chuyển CRS về UTM 48N và xóa cột fid → OK.

2. Cắt ảnh Raster (Clip Raster)

- Input layer: VLUCD_48N
- Target CRS: UTM 48N
- Assign: 0.00000
- Cuối cùng lưu ở file tif: HaNoi_48N_Clip.

3. Chuyển Raster sang Vector (Polygonize)

- Name of the field: classid
- Lưu ở file qpkg: HaNoi_48N_Polygon.

4. Tính diện tích và lưu mẫu

- Chuột phải file vừa tạo → chọn Open Attribute Table.
- Ở thanh công cụ kính lúp, tạo thêm cột mới tên area rồi gõ $area → chọn Update All.
- Chuột phải chọn Save Selected Feature As.
- Lưu ở file qpkg: 18_Open_Water_140K và xóa cột fid.

5. Xuất dữ liệu sang Google Earth (KML)

- Chuột phải file 18_Open_Water_140K → chọn Export → Save Feature As.
- Format: KML
- Lưu ở file kml: 18_Open_Water_gg_earth_pro.
- CRS: 4326 - WGS 84
